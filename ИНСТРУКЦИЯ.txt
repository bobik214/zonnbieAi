====================================================
  ПОДРОБНАЯ ИНСТРУКЦИЯ ПО ДЕПЛОЮ НА RENDER
====================================================

ШАГ 1: УСТАНОВКА GIT
--------------------
1. Скачай: https://git-scm.com/download/win
2. Установи (все настройки по умолчанию)
3. Перезапусти терминал/командную строку


ШАГ 2: ЗАГРУЗКА НА GITHUB
--------------------
Вариант А (автоматически):
   - Запусти файл upload-to-github.bat

Вариант Б (вручную):
   cd %USERPROFILE%\Desktop\ai-bot-js
   git init
   git add .
   git commit -m "Initial commit - AI Telegram Bot"
   git branch -M main

   Затем создай репозиторий на https://github.com/new
   Имя: ai-telegram-bot
   Public, без README

   git remote add origin https://github.com/ТВОЙ_НИК/ai-telegram-bot.git
   git push -u origin main


ШАГ 3: РЕГИСТРАЦИЯ НА RENDER
--------------------
1. Зайди на https://render.com
2. Нажми "Get Started" или "Sign Up"
3. Выбери "Sign in with GitHub"
4. Авторизуйся и дай доступ к репозиторию


ШАГ 4: СОЗДАНИЕ WEB SERVICE
--------------------
1. На дашборде Render нажми "New +"
2. Выбери "Web Service"
3. Найди свой репозиторий "ai-telegram-bot" и нажми "Connect"


ШАГ 5: НАСТРОЙКА SERVICE
--------------------
Заполни поля:

  Name:              ai-telegram-bot
  Region:            Frankfurt (EU) - ближайший к тебе
  Branch:            main
  Root Directory:    (оставь пустым)
  Runtime:           Node
  Build Command:     npm install
  Start Command:     node bot.js

Environment Variables (не обязательно, ключи уже в коде):
  BOT_TOKEN = 8541388104:AAECxzzmEiM0PbQOyozsj5JWIqLSdnI3Tlo
  OPENROUTER_API_KEY = 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


ШАГ 6: ЗАПУСК
--------------------
1. Нажми "Create Web Service"
2. Подожди 2-5 минут пока Render деплоит
3. Когда статус станет "Live" - бот работает!


ШАГ 7: ПРОВЕРКА
--------------------
1. Открой своего бота в Telegram
2. Напиши /start
3. Напиши .сколько лап у паука
4. Если ответил - всё работает!


====================================================
  ВОЗМОЖНЫЕ ПРОБЛЕМЫ
====================================================

Проблема: Render не находит репозиторий
Решение: Убедись что репозиторий Public, или дай доступ Render

Проблема: Бот не отвечает после деплоя
Решение: 
  - Проверь логи в Render (вкладка Logs)
  - Убедись что API ключи правильные

Проблема: Ошибка 404 от OpenRouter
Решение: Проверь что API ключ начинается с sk-or-v1-

Проблема: Бот работает только некоторое время
Решение: Бесплатный Render засыпает через 15 мин без запросов.
  Для 24/7 используй Railway.app или другой хостинг.


====================================================
  АЛЬТЕРНАТИВНЫЕ ХОСТИНГИ
====================================================

1. Railway (https://railway.app) - 500 часов/мес, не засыпает
2. Fly.io (https://fly.io) - 3 бесплатных VM
3. PythonAnywhere (https://pythonanywhere.com) - для Python версии
