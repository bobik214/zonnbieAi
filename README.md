# Telegram AI-бот с OpenRouter (Node.js версия)

ИИ-ассистент в Telegram с использованием бесплатных моделей через OpenRouter API.

## Возможности

- Ответы на вопросы с использованием ИИ
- Выбор из нескольких бесплатных AI моделей
- Админ-панель для управления API ключом (только для @tcpdog)
- Простое управление через inline-кнопки

## Бесплатные модели

- Gemma 2 9B (Google)
- Llama 3 8B (Meta)
- Mistral 7B
- OpenChat 7B
- OpenHermes 2.5
- Zephyr 7B

## Установка и запуск

### 1. Установите зависимости

```bash
npm install
```

### 2. Запустите бота

```bash
node bot.js
```

## Использование

### Обычный пользователь:

1. Откройте бота в Telegram
2. Напишите любой вопрос (например: "сколько лап у паука")
3. Получите ответ от ИИ
4. Используйте кнопки для выбора модели или получения помощи

### Администратор (@tcpdog):

1. Нажмите кнопку "Админ-панель"
2. Смените API ключ OpenRouter при необходимости
3. Просмотрите текущий API ключ

## Команды

- `/start` - Запустить бота
- `/help` - Помощь
- `/cancel` - Отменить текущее действие

## Технические детали

- **Runtime:** Node.js
- **Библиотека:** node-telegram-bot-api
- **API:** OpenRouter (совместим с OpenAI API)
- **Хранение настроек:** JSON файл
- **HTTP запросы:** node-fetch
